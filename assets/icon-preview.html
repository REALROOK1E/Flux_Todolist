🛠️ 图标转换步骤

  1. 在浏览器中右键点击上方的 512x512 图标
  2. 选择 "另存为图片" 或 "复制图片"
  3. 将图片保存为 PNG 格式 (建议命名为 icon-512.png)
  4. 使用下方的在线工具转换为所需格式
  5. 将转换后的文件放入 assets 目录
